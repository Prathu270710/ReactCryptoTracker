## Quick Start

```bash
# Clone the repo
git clone https://github.com/YourUsername/ReactCryptoTracker.git
cd ReactCryptoTracker

# Install dependencies
npm install

# Start the app
npm start
```

Open `http://localhost:3000` in your browser.

## Prerequisites

- Node.js (v14+)
- npm

## Usage

1. Browse live cryptocurrency prices
2. Use the search bar to find specific coins
3. View market cap, rankings, and 24h price changes
4. Green = gains, Red = losses
